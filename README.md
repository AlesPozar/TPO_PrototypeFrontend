# Opis
To je direktorij prototipa spletnega vmesnika naše aplikacije.

### Uporablja se
- Next.js
- ShadcnUI
- TailwindCSS

### Objava prototipa in morda kasnejših live verzij projekta
zaradi limitacij objave preko Vercel-a glede privatnih organizacij (Vercel hoče da mam pro plan, 🥀, mogoce aktiviram free-demo kasneje), sem se odločil, da bom kodo kopiral v druge, moje zunanje repozitorije in live objave delal preko njih.

Prototip izvira iz sledečega repozitorija:
https://github.com/AlesPozar/TPO_PrototypeFrontend
(public repo, da se lahko po potrebi spremljajo commiti)


# Navodila za lokalni zagon
### Potrebe
- npm / pnpm
### Navodila
1. si v root datoteki frontenda, tj, ***Projekt-26/src/prototip/TPO_PrototypeFrontend/***
2. V terminalu
    - npm install
    - npm run dev
3. Če je vse pravilno, se na koncu v terminalu izpiše ***localhost*** povezava do live(srvrčk) preview spletne strani.
